# 🎓 One-Stop Career & Education Advisor  

A **responsive web application** that helps students with **personalized career guidance** through an **aptitude test**, and provides details about **government colleges, their curriculum, and recommended courses**.  

🌍 Supports **multiple languages**: English, Hindi, Telugu, and Urdu.  

---

## 🚀 Features
- 📊 **Aptitude Test** with 10+ questions.  
- 🎯 **Personalized course recommendations** based on test results.  
- 🏫 **Government college listings** with curriculum details.  
- 📚 **Courses section** with specialization highlights.  
- 🌐 **Multilingual support** (English, Hindi, Telugu, Urdu).  
- 📱 Fully **responsive design** (mobile + desktop).
